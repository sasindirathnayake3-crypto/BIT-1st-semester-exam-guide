BIT 1st Semester Web Design Summary

Essential concepts: HTML Structure, Tables, and Hyperlinks

1. HTML Basics: Structure & Core Elements

HTML (HyperText Markup Language) provides the fundamental structure for every web page. It uses **elements** and **tags** to define content like headings, paragraphs, and lists.

Key Structural Tags

  • <!DOCTYPE html>: Defines the document type and version of HTML (must be the first line).
  • <html lang="en">: The root element of an HTML page, encapsulating all other content.
  • <head>: Contains meta-information about the HTML document (like the title, character set, and links to stylesheets).
  • <body>: Contains the visible page content, including text, images, links, tables, and media.

Common Content Tags

  • **Headings:** <h1> to <h6> for defining section hierarchy.
  • **Paragraphs:** <p> for standard text blocks.
  • **Formatting:** <strong> or <b> for bold text, <em> or <i> for emphasized/italic text.
  • **Line Break:** <br> (a self-closing tag for a carriage return).
  • **Divider:** <hr> (a self-closing tag for a horizontal rule).

2. HTML Tables: Organizing Tabular Data

The <table> element is used to present data in rows and columns. It's crucial for tabular data, but should **not** be used for general page layout.

Core Table Tags

  • <table>: The container for the entire table.
  • <tr>: Defines a table row.
  • <th>: Defines a table header cell (usually bold and centered).
  • <td>: Defines a standard table data cell.
  • <thead>, <tbody>, <tfoot>: Used for semantic grouping of content (Header, Body, Footer).

Span Attributes (Merging Cells)

Attribute Function Example
colspan Merges a cell across multiple columns horizontally. <td colspan="2">
rowspan Merges a cell down multiple rows vertically. <td rowspan="3">

3. Hyperlinks: The <a> Tag

The anchor element (<a>) is the core mechanism for navigation, linking one page to another, or one section to another within the same page.

Essential Attributes

  • **href (Hypertext Reference):** Specifies the destination URL (required). This can be an external website, a local file, or a section ID.
  • **target:** Specifies where to open the linked document.

Common target Values

_self    - Opens the link in the same window/tab (default).
_blank   - Opens the link in a new window/tab.
_parent  - Opens the link in the parent frame (if using frames).
_top     - Opens the link in the full body of the window.

Link Types Example

1. **External Link:** Google Search (Opens in new tab)
2. **Internal Link (to section):** Go to HTML Basics Section
3. **Email Link:** Send Email

Don't forget the **fragment identifier** (e.g., <a href="#top">) to jump to the top of the page!
